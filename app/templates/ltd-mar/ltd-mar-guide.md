# DIY Accounting Company Accounts User Guide

Thank you for using DIY Accounting as your limited company accounting system.

**Note:** This package supports the **small profits Corporation Tax rate** (19%) for companies with taxable profits up to £50,000. Companies with profits above £50,000 should seek professional advice regarding marginal relief and the main CT rate (25%).

Written upon Excel spreadsheets, the Company Accounts system automates bookkeeping through Excel formulae, producing monthly management accounts, published accounts, corporation tax calculations, and a CT600 company tax return.

## Sales Spreadsheet

Enter income in the Sales workbook. Each month has a separate sheet (Apr–Mar).

| Column | Entry |
|--------|-------|
| **A** | Date of sale |
| **B** | Customer name |
| **C** | Invoice number |
| **D** | Description |
| **E** | Code letter (A/B/C/D/G/O/FS) |
| **F** | Gross amount including VAT |

VAT and net amounts are calculated automatically in columns G and H.

## Purchases Spreadsheet

## Financial Accounts

The Financialaccounts workbook contains automated reports:

- **Trial Balance** — cumulative monthly balances
- **Monthly P&L** — management profit and loss account
- **Published P&L** — statutory format profit and loss
- **Published Balance Sheet** — statutory format balance sheet
- **Published Notes** — notes to the accounts
- **Corporation Tax** — CT calculation (small profits rate)
- **CT600** — company tax return summary
- **Wages Interface** — payroll integration
